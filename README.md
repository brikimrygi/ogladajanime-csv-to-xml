# Konwertuj plik .CSV z ogladajanime.pl, w plik .XML dla MyAnimeList.net 

---
1. Zainstaluj PYTHON3, który jest wymagany do działania skryptu!
---

```bash
Android: pkg install python3

Windows (Visual Studio Code Terminal): pkg install python3

Linux (Debian): sudo apt install python3

Linux (Arch): sudo pacman -S python3
```
---
2. Aby wprawić skrypt w ruch wpisz:
---

Musisz znajdować się w miejscu skryptu: "./skrypt/"!

```bash
python convert_anime_file.py
```

Po wpisaniu komendy, skrypt zapyta się ciebie o:
"Wprowadź ścieżkę do pliku .csv (możesz też przeciągnąć i upuścić plik w tym oknie):"
